## Supporting Evidence

- [ ] Firebase Authentication enabled in Firebase Console.
- [ ] Firestore collections visible in Firebase Console.
- [ ] Firebase Storage bucket visible in Firebase Console.
- [ ] Terminal showing `npm run typecheck` success.
- [ ] Terminal showing `npx expo export --platform android` success.
- [ ] README open in editor or preview.
- [ ] Database structure document open.

## Screenshot Quality Rules

- Use a readable mobile viewport.
- Avoid cropped headings or hidden buttons.
- Capture forms with realistic sample data where possible.
- Capture empty, loading, or error states only if useful for explanation.
- Do not show real Firebase secrets.
- Do not show personal passwords or private credentials.
