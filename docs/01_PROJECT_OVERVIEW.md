# 01 Project Overview

Authority: `PROJECT_BIBLE.md`

YUMMY Restaurant Mobile Application is a commercial-quality restaurant ordering product built for the Mobile Application Development assignment. The full product vision, mission, business goals, assignment goals, technical goals, scope, and out-of-scope boundaries are defined in `PROJECT_BIBLE.md`.

## Current Phase Status

- Phase 1: complete, architecture and planning.
- Phase 2: complete, Expo project setup and environment configuration.
- Phase 2.5: active, handbook and engineering standards.

## Product Summary

Customers will register, login, browse meals, search, add items to cart, checkout, submit dummy payment, track orders, and receive notifications. Administrators will login and manage foods, customers, orders, payments, and notifications.

Future phase work must begin by checking `PROJECT_BIBLE.md`, this overview, `DECISION_LOG.md`, and the relevant numbered standards document.
